Руководство по использованию оболочки "myshell"

ОПИСАНИЕ
---------------
myshell — это командная оболочка для операционных систем семейства UNIX.
Оболочка является посредником между пользователем и операционной системой. Она принимает команды, введённые пользователем, и передаёт их на исполнение.

Оболочка работает в двух режимах:
  1. Интерактивный режим: Вы вводите команды по одной и видите мгновенный результат.
  2. Пакетный режим: Оболочка читает и выполняет команды из заранее подготовленного файла.

ЗАПУСК ОБОЛОЧКИ
---------------
Интерактивный режим (работа в реальном времени):

    ./myshell

После запуска появится приглашение вида:

    myshell:/home/user>

Это означает, что оболочка готова принимать команды. Текст после двоеточия —
текущий рабочий каталог (папка, в которой вы сейчас находитесь).

Пакетный режим (выполнение команд из файла):

    ./myshell имя_файла

В этом режиме оболочка читает команды из указанного текстового файла и выполняет их последовательно, строка за строкой. Приглашение на экране
не отображается. Это удобно для автоматизации повторяющихся задач.

Пример файла с командами (batchfile):
    echo Начинаю работу
    cd /tmp
    dir
    echo Готово

Запуск: ./myshell batchfile


ВСТРОЕННЫЕ КОМАНДЫ
------------------
Следующие команды выполняются самой оболочкой без запуска внешних программ.

cd <каталог>
    Сменить текущий рабочий каталог на указанный.
    Пример:  cd /home/user/documents
    Если каталог не указан, выводится путь к текущему каталогу.

clr
    Очистить экран терминала.

dir <каталог>
    Показать содержимое указанного каталога (файлы и папки).
    Если каталог не указан, показывается содержимое текущего каталога.
    Пример:  dir /home/user

environ
    Вывести список всех переменных окружения и их значений.
    Пример вывода:
        HOME=/home/user
        PATH=/usr/bin:/bin
        SHELL=/home/user/myshell

echo <текст>
    Вывести указанный текст на экран.
    Пример:  echo Привет, мир!

help
    Показать краткий список доступных команд прямо в оболочке.

pause
    Приостановить выполнение до нажатия клавиши Enter.
    Удобно в пакетных файлах, когда нужно дать пользователю время
    прочитать вывод предыдущей команды.

exit или quit
    Завершить работу оболочки.


ВНЕШНИЕ КОМАНДЫ
---------------
Помимо встроенных команд, myshell может запускать любые программы,
установленные в системе. Для этого просто введите имя программы.

Примеры:
    ls -l          — подробный список файлов (стандартная команда UNIX)
    cat файл.txt   — вывод содержимого файла
    cp a.txt b.txt — копирование файла

Оболочка ищет программу в стандартных системных каталогах (PATH).
Если программа не найдена, выводится сообщение об ошибке.


ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ
--------------------
Переменные окружения — это именованные значения, которые операционная система
предоставляет всем запущенным программам. Их можно представить как глобальные
настройки системы.

Примеры важных переменных:

    PWD     — Путь к текущему рабочему каталогу. Автоматически обновляется командой cd.
    SHELL   — Полный путь к исполняемому файлу myshell. Устанавливается при запуске оболочки.


ЗАВЕРШЕНИЕ СЕАНСА
-----------------
Для выхода из оболочки введите:
    exit
или
    quit

Также можно нажать Ctrl+D — это сигнал конца ввода, оболочка завершится
автоматически.


ПРИМЕРЫ СЕАНСОВ РАБОТЫ
-----------------------
Пример 1: навигация по файловой системе
    ./myshell
    myshell:/home/user> cd /tmp
    myshell:/tmp> dir
    tempfile1.txt
    tempfile2.txt
    myshell:/tmp> cd
    /tmp
    myshell:/tmp>


ЗАВЕРШЕНИЕ РАБОТЫ ОБОЛОЧКИ
--------------------------

Для выхода из оболочки используйте один из следующих способов:

    1. Встроенная команда:
      a: myshell> quit
      b: myshell> exit
      
    2. Комбинация клавиш (в интерактивном режиме):
      Ctrl+D (сигнал конца файла)

    3. Автоматическое завершение:
       В пакетном режиме оболочка завершается после выполнения всех команд из файла.
